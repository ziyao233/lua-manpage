NAME

lua_closeslot

SYNOPSIS

#include <lua.h>

void lua_closeslot (lua_State *L, int index);

DESCRIPTION

Close the to-be-closed slot at the given index and set its value to @nil. The index must be the last index previously marked to be closed (see lua_toclose in Lua manual) that is still active (that is, not closed yet).

A __close metamethod cannot yield when called through this function.

ERRORS

This function can run arbitrary Lua code, any error could be raised.
