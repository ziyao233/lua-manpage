lua_rawgeti(3) Lua C API lua_rawgeti(3)

NAME

lua_rawgeti

SYNOPSIS

#include <lua.h>

int lua_rawgeti (lua_State *L, int index, lua_Integer n);

DESCRIPTION

Pushes onto the stack the value t[n] , where t is the table at the given index. The access is raw, that is, it does not use the __index metavalue.

Returns the type of the pushed value.

ERRORS

This function never raises any error.

STACK USAGE

Push
1
Pop
0
