luaL_unref(3) Lua C API luaL_unref(3)

NAME

luaL_unref

SYNOPSIS

#include <lauxlib.h>

void luaL_unref (lua_State *L, int t, int ref);

DESCRIPTION

Releases the reference ref from the table at index t (see luaL_ref(3)) . The entry is removed from the table, so that the referred object can be collected and the reference ref can be used again by luaL_ref .

If ref is LUA_NOREF or LUA_REFNIL , luaL_unref does nothing.

ERRORS

This function never raises any error.

0
