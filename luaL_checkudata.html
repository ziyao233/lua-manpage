NAME

luaL_checkudata

SYNOPSIS

#include <lauxlib.h>

void *luaL_checkudata (lua_State *L, int arg, const char *tname);

DESCRIPTION

Checks whether the function argument arg is a userdata of the type tname (see luaL_newmetatable(3)) and returns the userdata's memory-block address (see lua_touserdata(3)) .

ERRORS

Errors that could be raised are described in DESCRIPTION.

STACK USAGE

Push
0
Pop
0
