NAME

lua_Number

SYNOPSIS

#include <lua.h>

typedef /* ... */ lua_Number;

DESCRIPTION

The type of floats in Lua.

By default this type is double, but that can be changed to a single float or a long double. (See LUA_FLOAT_TYPE in luaconf.h
