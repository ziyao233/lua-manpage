lua_toboolean(3) Lua C API lua_toboolean(3)

NAME

lua_toboolean

SYNOPSIS

#include <lua.h>

int lua_toboolean (lua_State *L, int index);

DESCRIPTION

Converts the Lua value at the given index to a false C boolean value ( false 0 or 1 ). Like all tests in Lua, lua_toboolean returns true for any Lua value different from @false and @nil; otherwise it returns false. (If you want to accept only actual boolean values, use lua_isboolean to test the value's type.)

ERRORS

This function never raises any error.
