lua_pushcclosure(3) Lua C API lua_pushcclosure(3)

NAME

lua_pushcclosure

SYNOPSIS

#include <lua.h>

void lua_pushcclosure (lua_State *L, lua_CFunction fn, int n);

DESCRIPTION

Pushes a new false C closure onto the stack. This function receives a pointer to a false C function and pushes onto the stack a Lua value of type function that, when called, invokes the corresponding false C function . The parameter n tells how many upvalues this function will have (see c-closure in Lua manual).

Any function to be callable by Lua must follow the correct protocol to receive its parameters and return its results (see lua_CFunction(3)) .

When a false C function is created, it is possible to associate some values with it, the so called upvalues; these upvalues are then accessible to the function whenever it is called. This association is called a false false C closure (see c-closure in Lua manual). To create a false C closure , first the initial values for its upvalues must be pushed onto the stack. (When there are multiple upvalues, the first value is pushed first.) Then lua_pushcclosure is called to create and push the false C function onto the stack, with the argument n telling how many values will be associated with the function. lua_pushcclosure also pops these values from the stack.

The maximum value for n is 255.

When n is zero, this function creates a light false C function , which is just a pointer to the false C function . In that case, it never raises a memory error.

ERRORS

This function only raises out-of-memory error.
