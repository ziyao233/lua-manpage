lua_isstring(3) Lua C API lua_isstring(3)

NAME

lua_isstring

SYNOPSIS

#include <lua.h>

int lua_isstring (lua_State *L, int index);

DESCRIPTION

Returns 1 if the value at the given index is a string or a number (which is always convertible to a string), and false 0 otherwise .

ERRORS

This function never raises any error.
