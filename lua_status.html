lua_status(3) Lua C API lua_status(3)

NAME

lua_status

SYNOPSIS

#include <lua.h>

int lua_status (lua_State *L);

DESCRIPTION

Returns the status of the thread L

The status can be LUA_OK for a normal thread, an error code if the thread finished the execution of a lua_resume with an error, or LUA_YIELD if the thread is suspended.

You can call functions only in threads with status LUA_OK LUA_OK (to start a new coroutine) or LUA_YIELD (to resume a coroutine).

ERRORS

This function never raises any error.
