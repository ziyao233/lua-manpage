lua_numbertointeger(3) Lua C API lua_numbertointeger(3)

NAME

lua_numbertointeger

SYNOPSIS

#include <lua.h>

int lua_numbertointeger (lua_Number n, lua_Integer *p);

DESCRIPTION

Tries to convert a Lua float to a Lua integer; the float n must have an integral value. If that value is within the range of Lua integers, it is converted to an integer and assigned to *p

This macro may evaluate its arguments more than once.
