NAME

luaL_setmetatable

SYNOPSIS

#include <lauxlib.h>

void luaL_setmetatable (lua_State *L, const char *tname);

DESCRIPTION

Sets the metatable of the object on the top of the stack as the metatable associated with name tname in the registry (see luaL_newmetatable(3))

ERRORS

This function never raises any error.

STACK USAGE

Push
0
Pop
0
