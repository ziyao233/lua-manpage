NAME

luaL_addlstring

SYNOPSIS

#include <lauxlib.h>
void luaL_addlstring (luaL_Buffer *B, const char *s, size_t l);

DESCRIPTION

Adds the string pointed to by s with length l to the buffer B (see luaL_Buffer(3)) . The string can contain embedded zeros.

ERRORS

This function only raises out-of-memory error.

STACK USAGE

Push
unknown
Pop
unknown
