lua_error(3) Lua C API lua_error(3)

NAME

lua_error

SYNOPSIS

#include <lua.h>

int lua_error (lua_State *L);

DESCRIPTION

Raises a Lua error, using the value on the top of the stack as the error object. This function does a long jump, and therefore never returns (see luaL_error(3))

ERRORS

Errors that could be raised are described in DESCRIPTION.

STACK USAGE

Push
0
Pop
1
