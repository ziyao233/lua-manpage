luaL_buffaddr(3) Lua C API luaL_buffaddr(3)

NAME

luaL_buffaddr

SYNOPSIS

#include <lauxlib.h>

char *luaL_buffaddr (luaL_Buffer *B);

DESCRIPTION

Returns the address of the current content of buffer B (see luaL_Buffer(3))

ERRORS

This function never raises any error.

STACK USAGE

Push
0
Pop
0
