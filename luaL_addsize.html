NAME

luaL_addsize

SYNOPSIS

#include <lauxlib.h>

void luaL_addsize (luaL_Buffer *B, size_t n);

DESCRIPTION

Adds to the buffer B a string of length n previously copied to the buffer area (see luaL_prepbuffer(3)) .

ERRORS

This function never raises any error.

STACK USAGE

Push
unknown
Pop
unknown
