lua_tonumberx(3) Lua C API lua_tonumberx(3)

NAME

lua_tonumberx

SYNOPSIS

#include <lua.h>
lua_Number lua_tonumberx (lua_State *L, int index, int *isnum);

DESCRIPTION

Converts the Lua value at the given index to the C type lua_Number (see lua_Number(3)) . The Lua value must be a number or a string convertible to a number (see coercion in Lua manual); otherwise, lua_tonumberx returns 0.

If isnum is not NULL , its referent is assigned a boolean value that indicates whether the operation succeeded.

ERRORS

This function never raises any error.

0
Pop
0
